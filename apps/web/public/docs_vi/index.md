# Bắt Đầu Nhanh (Getting Started)

Chào mừng bạn đến với tài liệu hướng dẫn P2P Digital Data Marts! Nền tảng này giúp bạn thu thập dữ liệu (ETL) từ hơn 200+ nguồn (Facebook, Google, Shopify...) về Data Warehouse của bạn như BigQuery, PostgreSQL hoàn toàn miễn phí và mã nguồn mở.

## Hướng dẫn 3 bước cơ bản:
1. **Thiết lập Nơi lưu trữ (Storage):** Vào menu *Storage*, điền thông tin kho dữ liệu của bạn. Đây là nơi tất cả dữ liệu từ quảng cáo, CRM sẽ được đổ về. 
2. **Kích hoạt Nguồn (Connectors):** Vào *Data Marts*, ấn tạo mới và chọn nguồn bạn cần kết nối (ví dụ: Facebook Ads). Cập nhật API Keys và các cấu hình liên quan.
3. **Cấu hình Đích đến (Destination):** Thiết lập việc xuất dữ liệu đã tinh chỉnh nếu cần, hoặc sử dụng trực tiếp BI tool nối với Storage.

> Mọi dữ liệu đi trực tiếp từ nguồn tới máy chủ P2P tự quản lý của bạn mà không hề qua trung gian. Không ai thu thập dữ liệu ngoài chính bạn!

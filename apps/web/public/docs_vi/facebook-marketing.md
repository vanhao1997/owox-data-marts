# Kết Nối Facebook Marketing

Tài liệu này hướng dẫn cách kết nối P2P Data Marts với hệ thống Facebook Ads.

### Lấy Token từ Facebook Developer:
1. Vào [Meta for Developers](https://developers.facebook.com/).
2. Đăng nhập và nhấp vào biểu tượng **Tạo ứng dụng (Create App)** -> Loại **Business**.
3. Chọn **Marketing API**.
4. Sử dụng công cụ **Graph API Explorer** để gen Token với các quyền: `ads_read`, `read_insights`, `ads_management`.

*(Cấu trúc tài liệu chi tiết có thể tiếp tục mở rộng. Vui lòng sử dụng tính năng Automation Script để tự động update Markdown bản dịch)*.
